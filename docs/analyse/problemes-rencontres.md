# Problèmes rencontrés

Comme tout projet technique réel, YTech Solutions a rencontré plusieurs difficultés. Ces problèmes ont concerné aussi bien le déploiement que la sécurité, les outils de supervision ou certaines contraintes système.

## Exemples de difficultés rencontrées

Parmi les principaux problèmes identifiés :

- manque d’espace disque ;
- conflits de ports entre services ;
- problèmes de plugins Nessus obsolètes ;
- difficulté d’authentification dans Graylog ;
- comportements réseau ou système limitant certaines protections ;
- difficultés sur certaines solutions d’accès distant.

## Intérêt pédagogique

Ces incidents ont eu une réelle valeur pédagogique. Ils ont obligé l’équipe à :

- diagnostiquer des situations concrètes ;
- tester plusieurs approches ;
- adapter certaines décisions techniques ;
- documenter précisément les solutions appliquées.

Autrement dit, les problèmes rencontrés ne sont pas uniquement des obstacles : ils font aussi partie de l’apprentissage et renforcent la crédibilité du projet.
