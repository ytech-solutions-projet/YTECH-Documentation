# Résultats obtenus

Le projet YTech Solutions a permis d’obtenir plusieurs résultats concrets sur les plans technique, organisationnel et sécuritaire.

## Résultats techniques

- déploiement d’un site vitrine fonctionnel ;
- déploiement d’une application RH opérationnelle ;
- intégration d’un backend exécuté avec PM2 ;
- mise en place d’un reverse proxy avec Nginx ;
- présence d’outils de supervision, de journalisation et de gestion des secrets.

## Résultats en matière de sécurité

- amélioration de l’organisation globale de l’architecture ;
- mise en place d’un pare-feu OPNsense avec segmentation et règles de filtrage ;
- meilleure maîtrise des accès ;
- renforcement du serveur Linux ;
- visibilité accrue grâce à Zabbix et Graylog ;
- scans de vulnérabilités réalisés avec Nessus ;
- gestion plus rigoureuse des secrets avec Bitwarden.

## Résultats pédagogiques

Le projet a également permis de développer plusieurs compétences :

- compréhension des architectures web ;
- administration Linux ;
- supervision ;
- gestion des accès ;
- journalisation ;
- analyse de sécurité ;
- documentation technique.

Le résultat global est donc positif : l’environnement final est plus structuré, mieux observé et plus cohérent que l’environnement de départ.
