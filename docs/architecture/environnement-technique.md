# Environnement technique

Le projet YTech Solutions repose sur plusieurs technologies et outils complémentaires.

## Technologies applicatives

- **React** : construction de l’interface utilisateur ;
- **Vite** : environnement de développement et de build frontend ;
- **Node.js** : exécution JavaScript côté serveur ;
- **Express** : framework backend ;
- **PostgreSQL** : base de données relationnelle.

## Hébergement et administration

- **Ubuntu Server** : système d’exploitation du serveur ;
- **Nginx** : reverse proxy ;
- **PM2** : gestionnaire de processus Node.js ;
- **Docker** : déploiement de certains outils.

## Outils de sécurité et de supervision

- **Fail2Ban** ;
- **Graylog** ;
- **Zabbix** ;
- **Bitwarden / Vaultwarden** ;
- **Nessus Essentials** ;
- **Tailscale** ;
- **Ngrok** ;
- **WAF** en démonstration.

L’ensemble de ces outils montre que le projet ne se limite pas à une simple application. Il s’agit d’un environnement technique complet, articulé autour de plusieurs briques interdépendantes.
