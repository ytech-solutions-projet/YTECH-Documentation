# Équipe et répartition des tâches

Le projet YTech Solutions a été réalisé en équipe. La répartition des tâches a permis de structurer le travail selon les rôles et les domaines techniques concernés.

## Othmane BENAMAR

Les missions principales ont porté sur :

- le déploiement et la sécurisation de l’application web ;
- la configuration du reverse proxy Nginx ;
- la mise en place du HTTPS / TLS ;
- le hardening du serveur web ;
- les tests techniques et de sécurité ;
- la résolution de problèmes liés au proxy, aux en-têtes, aux accès et au flux HTTPS ;
- la participation à la documentation et à la présentation.

## Achraf Hmichane

Les contributions principales comprennent :

- l’analyse de l’infrastructure initiale ;
- l’identification des failles ;
- la proposition d’une architecture sécurisée ;
- la mise en place d’une architecture avec DMZ et segmentation ;
- la mise en place du serveur Linux ;
- le déploiement de l’application CRUD RH ;
- la mise en place d’un reverse proxy ;
- l’intégration d’un WAF en démonstration ;
- le hardening du serveur Linux ;
- la sécurisation SSH ;
- l’organisation du projet avec Jira.

## Mostafa NASIR

Les tâches principales comprennent :

- l’organisation du projet sur GitHub ;
- la gestion des tâches avec Trello ;
- la mise en place de Nessus ;
- la mise en place de Bitwarden / Vaultwarden ;
- la mise en place de Zabbix ;
- la mise en place de Graylog.

## Ismail ELOUAKIFI

Les contributions portent principalement sur :

- la conception de l’architecture initiale sur Packet Tracer ;
- la conception de l’architecture finale sur GNS3 ;
- l’élaboration du plan d’adressage réseau.

## Bouchaib HERRADI

Les tâches comprennent :

- la mise en place de Jenkins ;
- l’utilisation de Trivy ;
- la mise en place de SonarQube ;
- la gestion des images sur Docker Hub ;
- les notifications par e-mail ;
- la participation à Prometheus et Grafana.

Cette répartition montre que le projet ne relève pas d’un seul domaine. Il mobilise à la fois des compétences en développement, administration système, réseau, cybersécurité et organisation de projet.
