# Introduction

Dans le cadre du projet **YTech Solutions**, notre objectif principal a été de concevoir, déployer et sécuriser une infrastructure capable d’héberger une application web dans de bonnes conditions de fonctionnement et avec un niveau de sécurité cohérent. Contrairement à un simple projet de développement web, ce travail ne se limite pas à la création d’une interface ou d’une API. Il inclut également la structuration de l’environnement d’hébergement, l’organisation des services, la gestion des accès, la supervision, la journalisation et l’analyse de la sécurité.

Pour une personne qui ne maîtrise pas encore le domaine de l’informatique, il faut comprendre qu’une application web n’est que la partie visible d’un ensemble plus large. Lorsqu’un utilisateur ouvre un site ou une application dans son navigateur, plusieurs composants travaillent ensemble en arrière-plan : un serveur web, un service applicatif, une base de données, parfois un proxy, et divers outils destinés à surveiller ou protéger l’ensemble.

Dans ce projet, nous avons donc cherché à aller au-delà du simple fonctionnement technique. Nous avons analysé l’architecture initiale, identifié ses limites, proposé une architecture plus sécurisée, puis mis en place différentes briques techniques afin de mieux contrôler les accès, améliorer la visibilité sur le système et réduire la surface d’exposition.

Le projet a également permis de mettre en pratique plusieurs dimensions complémentaires :

- le déploiement d’une application web ;
- l’administration d’un serveur Linux ;
- la mise en place d’un reverse proxy ;
- la sécurisation des accès ;
- la supervision et la centralisation des journaux ;
- l’analyse de vulnérabilités ;
- la documentation technique des choix réalisés.

Ainsi, YTech Solutions constitue un projet transversal reliant développement, infrastructure, cybersécurité et méthodologie. Cette documentation a pour but d’expliquer ces différents aspects de manière claire, structurée et accessible, y compris pour un lecteur peu familier avec les notions techniques.
