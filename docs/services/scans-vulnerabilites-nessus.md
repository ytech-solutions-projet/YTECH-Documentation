# Scans de vulnérabilités avec Nessus

La sécurité ne peut pas être évaluée uniquement à l’intuition. Il est important de vérifier concrètement si un environnement présente encore des faiblesses. Dans le projet YTech Solutions, cette vérification a été réalisée à l’aide de **Nessus Essentials**.

## Intérêt

Les scans de vulnérabilités permettent :

- d’identifier des points faibles ;
- de repérer certaines mauvaises configurations ;
- de guider les actions de correction ;
- de documenter plus concrètement la posture de sécurité du système.

Pour un lecteur non spécialiste, on peut considérer Nessus comme un outil de contrôle technique appliqué à l’environnement informatique.
